# Calm Check v1.2

Calm Check — PWA для спокойных сценарных проверок, помогающее снизить неопределённость и перестать возвращаться к одним и тем же мыслям после того, как проверка уже выполнена.

Приложение помогает один раз пройти заранее подготовленный сценарий, сохранить факт его выполнения и двигаться дальше без повторных проверок.

## Возможности

- Создание собственных сценариев проверки
- Пошаговое прохождение с чек-листом
- Автоматическое сохранение истории
- Статистика использования
- Закрепление важных сценариев
- Экспорт и импорт данных
- Работа полностью офлайн
- Установка как PWA
- Минималистичный интерфейс без лишних отвлекающих элементов
